Media Queries

반응형 웹 사이트를 만들기 위한 것
각 미디어 매체(screen, print)에 따라 다른 스타일 시트를 적용할 수 있게 만드는 것!
css2에서는 media type

Media Queries 구문

@media media queries{/*style rules*/}
media queries가 참이면 적용!

  • 미디어 타입 : all, print, screen
  • 미디어 특성 : width, orientation

Syntax

S* : 공백
[] : 있을 수 도 없을 수 도
a|b : a or b
? : 0 or 1
* : 0 or 1 or ... n

media_query_list
: S* [media_query [ ',' S* media_query ]* ]?
;
media_query
: [ONLY | NOT]? S* media_type S* [ AND S* expression ]*
| expression [ AND S* expression ]*
;
expression
: '(' S* media_feature S* [ ':' S* expr ]? ')' S*
;

  1. 여러 개의 미디어 쿼리로 이루어진 리스트로 작성할 수 있다.
  2. 미디어 타입 또는 표현식 단독으로도 작성할 수 있다.
  3. 표현식은 값이 없이 특성 이름만으로 작성할 수 있다.

/* mobile first*/
body { background-color: gold; }
@media (min-width: 768px) and (max-width: 1024px) {
body { background-color: lightblue; }
}
@media (min-width: 1025px) {
body { background-color: lightpink; }
}

/* desktop first*/
body { background-color: lightpink; }
@media (min-width: 768px) and (max-width: 1024px) {
body { background-color: lightblue; }
}
@media (max-width: 767px) {
body { background-color: gold; }
}
